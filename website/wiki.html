Datapack Documentation

Create content for Omega without writing code using JSON Datapacks.

File Structure

To add custom content, create a Datapack with the following directory structure inside your data/ folder.

Datapack Directory Tree
data/
  omega/
    omega_clans/              # Custom Clans
      your_clan.json
    omega_clan_specializations/
      your_spec.json
    omega_souls/              # Custom Soul Shards
      your_soul.json
    afo_mappings/             # All For One configurations
      techniques.json

JSON Specifications

Clan JSON

Place in data/omega/omega_clans/. Ensure the file name matches the ID if possible.

example_clan.json 
{
  "id": "mugen",
  "display_name": "Mugen Clan",
  "color": "#00FFFF",
  "has_god_ki": true,
  "specializations": ["Sage", "Cataclysm"],
  "default_traits": ["Angel"], 
  "simple_domain_radius_mult": 1.25,
  "rct_efficiency_mult": 1.5,
  "technique_cost_mult": 0.8,
  "bonus_health": 10.0,
  "bonus_speed": 0.05
}

Soul JSON

Place in data/omega/omega_souls/.

example_soul.json 
{
  "id": "ancient_sorcerer",
  "name": "Ancient Sorcerer",
  "color": "#FF0000",
  "blip_sound": "minecraft:entity.experience_orb.pickup",
  "damage_multiplier": 1.2,
  "dialogue": {
    "COMBAT_START": ["Show me your strength.", "I have awoken."],
    "PLAYER_LOW_HEALTH": ["Do not falter now."]
  }
}

JSON Attribute Reference

This table lists all available keys for use in Clan, Specialization, and Soul JSON files. Not all keys are applicable to every type.

Key Type Default Description
Identity & Display
idStringN/AThe unique technical ID (e.g., "zenin"). Required.
display_nameStringIDThe in-game display name (e.g., "Zenin Clan"). For Souls, this is `name`.
colorString"#FFFFFF"Hex color code for clan/soul name display.
has_god_kiBooleanfalseIf true, the entity cannot be sensed by Six Eyes unless the viewer also has God Ki.
Clan Features
specializationsList (String)[](Clan only) A list of Specialization IDs available to this clan.
default_traitsList (String)[](Clan only) A list of Trait IDs granted when this clan is acquired.
default_physical_traitsList (String)[](Clan only) A list of Physical Trait IDs granted when this clan is acquired.
Soul Audio & Dialogue
blip_soundString"omega:maleblip"(Soul only) Sound event ID for dialogue text.
blip_pitchNumber1.0(Soul only) Pitch for the blip sound.
blip_volumeNumber1.0(Soul only) Volume for the blip sound.
secondary_blip_soundStringnull(Soul only) Optional second sound for a "fused" voice effect.
dialogueObjectnull(Soul only) An object mapping Dialogue Triggers to lists of strings.
Jujutsu Modifiers
damage_multiplierNumber1.0Multiplier for unarmed and technique damage.
weapon_damage_multiplierNumber1.0Multiplier for weapon damage. Stacks with damage_multiplier.
technique_cost_multNumber1.0Multiplier for Cursed Energy costs. Lower is better.
domain_expansion_multNumber1.0Multiplier for Domain Expansion power/refinement.
simple_domain_radius_multNumber1.0Multiplier for Simple Domain's radius.
simple_domain_level_bonusNumber0.0Flat bonus added to Simple Domain's output level.
da_drain_multNumber1.0Multiplier for Domain Amplification's CE drain. Lower is better.
da_level_bonusNumber0.0Flat bonus added to Domain Amplification's output level.
rct_efficiency_multNumber1.0Multiplier for Reverse Cursed Technique healing amount. Higher is better.
rct_fatigue_multNumber1.0Multiplier for fatigue gained from using RCT. Lower is better.
guard_damage_multNumber1.0Multiplier for damage taken while guarding. Lower is better.
guard_duration_multNumber1.0Multiplier for how long a guard can be held.
black_flash_chance_bonusNumber0.0Additive bonus to Black Flash chance (0.05 = +5%).
Stat Bonuses
bonus_healthNumber0(Clans/Specs) Flat bonus to max health (2 = 1 heart).
health_bonusNumber0(Souls) Flat bonus to max health (2 = 1 heart).
bonus_armorNumber0(Clans/Specs) Flat bonus to armor points.
armor_bonusNumber0(Souls) Flat bonus to armor points.
bonus_attack_damageNumber0(Clans/Specs) Flat bonus to base attack damage.
attack_bonusNumber0(Souls) Flat bonus to base attack damage.
bonus_speedNumber0.0(Clans/Specs) Additive bonus to movement speed (0.1 = +10%).
speed_bonusNumber0.0(Souls) Additive bonus to movement speed (0.1 = +10%).
bonus_knockback_resistanceNumber0.0(Clans/Specs) Additive bonus to knockback resistance.

Dialogue Triggers

Use these keys within the dialogue object in Soul JSONs to trigger chat messages.

Trigger KeyDescription
COMBAT_STARTFires when the player enters combat (takes damage or deals damage after being idle).
PLAYER_LOW_HEALTHFires when player HP drops below 30%.
TARGET_LOW_HEALTHFires when the opponent's HP drops below 20%.
PLAYER_KILL_MOBFires on killing a standard mob.
PLAYER_KILL_BOSSFires on killing a boss (Wither/Ender Dragon).
PLAYER_DEATHFires when the player dies.
ENTER_NETHERFires when entering the Nether dimension.
ENTER_ENDFires when entering the End dimension.
IDLEFires rarely when standing still for a long time.
NIGHT_TIMEFires occasionally at night.
